Paso 1

¿Qué vamos a construir?

Un formulario Android con Kotlin y Jetpack Compose que guarda registros de personas en memoria (sin base de datos).

📱

Jetpack Compose

Framework moderno de Google para crear interfaces Android con código declarativo en Kotlin.

🧠

Estado (State)

Los datos que la UI observa y actualiza automáticamente cuando cambian.

🗂️

Memoria (RAM)

Los datos se guardan en una lista mientras la app está abierta. Al cerrarla se borran.

✅

Validación

Verificamos que los campos no estén vacíos y que el formato sea correcto antes de guardar.

🗂️ Estructura del proyecto

  • 📄 model/Persona.kt — Clase de datos (nombre, email, edad…)
  • 📄 ui/RegistroScreen.kt — Pantalla principal con el formulario
  • 📄 MainActivity.kt — Punto de entrada de la app
💡

Concepto clave: En Compose, la UI se redibuja automáticamente cada vez que un estado cambia. No necesitas llamar a invalidate() ni notifyDataSetChanged() como en el sistema antiguo.

Paso 2

Modelo de datos

Primero definimos la estructura de los datos que vamos a guardar usando una data class.

¿Qué es una data class?

Es una clase especial de Kotlin diseñada para guardar datos. Kotlin genera automáticamente equals(), hashCode(), toString() y copy().

model/Persona.kt 
// El paquete indica dónde vive este archivo
package com.tuapp.registro.model

// data class = clase especial para guardar datos
data class Persona(
    val id: Int,         // identificador único
    val nombre: String,   // texto — nombre de la persona
    val apellido: String, // texto — apellido
    val email: String,    // texto — correo electrónico
    val telefono: String, // texto — número de teléfono
    val edad: Int         // número — edad en años
)

¿Por qué val dentro de la data class?

  • Cada campo usa val porque un objeto Persona no debería cambiar una vez creado.
  • Si quisieras modificar una persona, usarías .copy(nombre = "nuevo") en lugar de mutarla.
  • Esto se llama inmutabilidad — un principio importante en Kotlin.

Paso 3

val vs var — La regla de oro

Esta es la diferencia más importante en Kotlin. Entenderla bien evita la mayoría de errores en formularios.

❌ val — no reasignable
val nombre = "José"
nombre = "Carlos"
// ERROR: val no puede
// cambiar su referencia
✅ var — reasignable
var nombre = "José"
nombre = "Carlos"
// OK: var sí puede
// cambiar su valor

La caja sellada (analogía)

Imagina que val es una caja sellada con candado. No puedes cambiar la caja, pero si la caja tiene compartimentos internos, puedes cambiar lo que hay dentro de ellos.


Por eso val nombre = remember { mutableStateOf("") } funciona: la caja (referencia) es fija, pero el contenido (.value) sí puede cambiar.

Declaración¿Puedes hacer x = "algo"?Usar en formularios
val x = ... ❌ No Solo con .value
var x = ... ✅ Sí Directo sin .value
val x by remember {...} ❌ No — ERROR Combinación inválida para campos
var x by remember {...} ✅ Sí La forma más limpia
⚠️

Error común: Usar val con by es la combinación que da error en limpiar(). Siempre usa var cuando necesites reasignar el campo directamente.

Paso 4

mutableStateOf y remember

Así declaramos los campos del formulario para que Compose los observe y redibuje la pantalla automáticamente.

¿Qué hace remember?

remember guarda el valor en memoria mientras el Composable está activo. Sin él, el estado se reiniciaría en cada redibujado.

¿Qué hace mutableStateOf?

mutableStateOf crea un contenedor observable. Cuando su valor cambia, Compose sabe que debe redibujar los componentes que lo usan.

✅ Opción A — var + by (recomendada)
// Declarar
var nombre by remember {
    mutableStateOf("")
}

// Leer y escribir SIN .value
Text(nombre)
nombre = "José"
✅ Opción B — val + = + .value
// Declarar
val nombre = remember {
    mutableStateOf("")
}

// Leer y escribir CON .value
Text(nombre.value)
nombre.value = "José"
Declaración de estados del formulario 
// ── Campos del formulario (var + by = sin .value) ──
var nombre   by remember { mutableStateOf("") }
var apellido by remember { mutableStateOf("") }
var email    by remember { mutableStateOf("") }
var telefono by remember { mutableStateOf("") }
var edad     by remember { mutableStateOf("") }

// ── Errores de validación (Boolean) ──
var nombreError   by remember { mutableStateOf(false) }
var apellidoError by remember { mutableStateOf(false) }
var emailError    by remember { mutableStateOf(false) }
var telefonoError by remember { mutableStateOf(false) }
var edadError     by remember { mutableStateOf(false) }

// ── Lista en memoria ──
val personas = remember { mutableStateListOf<Persona>() }
var nextId   by remember { mutableIntStateOf(1) }

// ── Snackbar (NO va dentro de mutableStateOf) ──
val snackbarHostState = remember { SnackbarHostState() }
💡

Nota: SnackbarHostState no es un estado observable de Compose, es un objeto de control. Por eso va directo en remember { SnackbarHostState() }, sin mutableStateOf.

Paso 5

Funciones: limpiar, validar y guardar

Funciones locales dentro del Composable que manipulan el estado del formulario.

Función limpiar() 
// Resetea todos los campos y errores al estado inicial
fun limpiar() {
    nombre   = ""  // limpia el campo nombre
    apellido = ""  // limpia el campo apellido
    email    = ""  // limpia el campo email
    telefono = ""  // limpia el campo teléfono
    edad     = ""  // limpia el campo edad

    // También limpiamos los indicadores de error
    nombreError   = false
    apellidoError = false
    emailError    = false
    telefonoError = false
    edadError     = false
}
Función validar() 
// Retorna true si todo está correcto, false si hay errores
fun validar(): Boolean {
    var valido = true

    // Verificamos que nombre no esté vacío
    if (nombre.isBlank()) {
        nombreError = true   // activa el mensaje de error
        valido = false        // marca el formulario como inválido
    }

    // Verificamos apellido
    if (apellido.isBlank()) {
        apellidoError = true
        valido = false
    }

    // Verificamos email con formato válido
    if (email.isBlank() || !android.util.Patterns.EMAIL_ADDRESS
            .matcher(email).matches()) {
        emailError = true
        valido = false
    }

    // Verificamos que edad sea un número válido
    val edadNum = edad.toIntOrNull()
    if (edadNum == null || edadNum !in 1..120) {
        edadError = true
        valido = false
    }

    return valido  // true = todo OK, false = hay errores
}
Función guardar() 
fun guardar() {
    // Si la validación falla, detenemos la ejecución
    if (!validar()) return

    // Creamos el objeto Persona con los datos del formulario
    personas.add(
        Persona(
            id       = nextId++,          // incrementa el ID
            nombre   = nombre.trim(),    // .trim() quita espacios extra
            apellido = apellido.trim(),
            email    = email.trim().lowercase(),  // minúsculas
            telefono = telefono.trim(),
            edad     = edad.toInt()     // ya validamos que es número
        )
    )

    limpiar()  // limpia el formulario después de guardar
}
💡

isBlank() vs isEmpty(): isEmpty() solo verifica si la cadena tiene longitud 0. isBlank() también considera espacios en blanco como vacío. Para formularios, isBlank() es mejor.

Paso 6

OutlinedTextField

El componente principal para capturar texto en Jetpack Compose con Material 3.

Campo de nombre con validación 
OutlinedTextField(
    // Muestra el valor actual del estado
    value = nombre,

    // Se llama cada vez que el usuario escribe algo
    // 'it' es el nuevo texto ingresado
    onValueChange = {
        nombre = it          // actualiza el estado
        nombreError = false  // quita el error mientras escribe
    },

    // Etiqueta flotante del campo
    label = { Text("Nombre *") },

    // Ícono al inicio del campo
    leadingIcon = {
        Icon(Icons.Default.Person, contentDescription = null)
    },

    // true = muestra el campo en rojo con mensaje de error
    isError = nombreError,

    // Texto que aparece debajo cuando hay error
    supportingText = if (nombreError) {
        { Text("El nombre es obligatorio") }
    } else null,

    // Ocupa todo el ancho disponible
    modifier = Modifier.fillMaxWidth(),

    // No permite múltiples líneas
    singleLine = true
)

Tipos de teclado con KeyboardOptions

  • KeyboardType.Text — teclado normal (por defecto)
  • KeyboardType.Email — teclado con @ y .com
  • KeyboardType.Number — solo números
  • KeyboardType.Phone — teclado de teléfono
Campo de email con teclado especial 
OutlinedTextField(
    value = email,
    onValueChange = { email = it; emailError = false },
    label = { Text("Correo electrónico *") },
    leadingIcon = {
        Icon(Icons.Default.Email, contentDescription = null)
    },
    isError = emailError,
    supportingText = if (emailError) {
        { Text("Ingresa un correo válido") }
    } else null,
    modifier = Modifier.fillMaxWidth(),
    singleLine = true,

    // Activa el teclado de email en el dispositivo
    keyboardOptions = KeyboardOptions(
        keyboardType = KeyboardType.Email
    )
)
💡

RoundedCornerShape: Se usa para darle esquinas redondeadas a botones y tarjetas. RoundedCornerShape(50) hace un botón en forma de píldora. El import necesario es androidx.compose.foundation.shape.RoundedCornerShape.

Paso 7

Lista en memoria

Guardamos los registros en una lista observable. Cuando agregamos o eliminamos, la UI se actualiza sola.

mutableStateListOf vs mutableListOf

  • mutableListOf() — lista normal de Kotlin, Compose NO la observa
  • mutableStateListOf() — lista observable, Compose SÍ la observa y redibuja cuando cambia
Lista observable y LazyColumn 
// Lista que Compose observa automáticamente
val personas = remember { mutableStateListOf<Persona>() }

// LazyColumn = RecyclerView moderno en Compose
// Solo renderiza los elementos visibles en pantalla
LazyColumn(
    modifier = Modifier.fillMaxSize(),
    verticalArrangement = Arrangement.spacedBy(8.dp),
    contentPadding = PaddingValues(vertical = 16.dp)
) {

    // Elemento único (tarjeta del formulario)
    item {
        FormularioCard()
    }

    // Lista de personas — key evita redibujos innecesarios
    items(personas, key = { it.id }) { persona ->
        PersonaCard(
            persona = persona,
            onEliminar = { personas.remove(persona) }
        )
    }

    // Muestra mensaje si la lista está vacía
    if (personas.isEmpty()) {
        item {
            Text(
                "Sin registros aún",
                color = MaterialTheme.colorScheme.onSurfaceVariant
            )
        }
    }
}
Tarjeta de persona con avatar 
@Composable
fun PersonaCard(persona: Persona, onEliminar: () -> Unit) {

    // Genera iniciales: "José Vinicio" → "JV"
    val iniciales = buildString {
        if (persona.nombre.isNotEmpty())
            append(persona.nombre.first().uppercaseChar())
        if (persona.apellido.isNotEmpty())
            append(persona.apellido.first().uppercaseChar())
    }

    Card(modifier = Modifier.fillMaxWidth()) {
        Row(
            modifier = Modifier.padding(12.dp),
            verticalAlignment = Alignment.CenterVertically
        ) {
            // Círculo con iniciales
            Box(
                modifier = Modifier
                    .size(44.dp)
                    .clip(CircleShape)
                    .background(MaterialTheme.colorScheme.primaryContainer),
                contentAlignment = Alignment.Center
            ) {
                Text(iniciales, fontWeight = FontWeight.Bold)
            }

            Spacer(Modifier.width(12.dp))

            // Datos de la persona
            Column(modifier = Modifier.weight(1f)) {
                Text("${persona.nombre} ${persona.apellido}")
                Text(persona.email, style = MaterialTheme.typography.bodySmall)
            }

            // Botón eliminar
            IconButton(onClick = onEliminar) {
                Icon(Icons.Default.Delete, tint = MaterialTheme.colorScheme.error)
            }
        }
    }
}
Paso 8

Código completo

El archivo RegistroScreen.kt completo con todos los conceptos integrados.

ui/RegistroScreen.kt — Completo 
package com.tuapp.registro.ui

import androidx.compose.foundation.background
import androidx.compose.foundation.layout.*
import androidx.compose.foundation.lazy.LazyColumn
import androidx.compose.foundation.lazy.items
import androidx.compose.foundation.shape.CircleShape
import androidx.compose.foundation.shape.RoundedCornerShape
import androidx.compose.foundation.text.KeyboardOptions
import androidx.compose.material.icons.Icons
import androidx.compose.material.icons.filled.*
import androidx.compose.material3.*
import androidx.compose.runtime.*
import androidx.compose.ui.Alignment
import androidx.compose.ui.Modifier
import androidx.compose.ui.draw.clip
import androidx.compose.ui.text.font.FontWeight
import androidx.compose.ui.text.input.KeyboardType
import androidx.compose.ui.unit.dp
import com.tuapp.registro.model.Persona

@OptIn(ExperimentalMaterial3Api::class)
@Composable
fun RegistroScreen() {

    // ── ESTADO DEL FORMULARIO ──────────────────────
    var nombre   by remember { mutableStateOf("") }
    var apellido by remember { mutableStateOf("") }
    var email    by remember { mutableStateOf("") }
    var telefono by remember { mutableStateOf("") }
    var edad     by remember { mutableStateOf("") }

    // ── ERRORES DE VALIDACIÓN ──────────────────────
    var nombreError   by remember { mutableStateOf(false) }
    var apellidoError by remember { mutableStateOf(false) }
    var emailError    by remember { mutableStateOf(false) }
    var telefonoError by remember { mutableStateOf(false) }
    var edadError     by remember { mutableStateOf(false) }

    // ── LISTA EN MEMORIA ───────────────────────────
    val personas = remember { mutableStateListOf<Persona>() }
    var nextId   by remember { mutableIntStateOf(1) }

    // ── SNACKBAR ───────────────────────────────────
    val snackbarHostState = remember { SnackbarHostState() }
    val scope = rememberCoroutineScope()

    // ── FUNCIÓN LIMPIAR ────────────────────────────
    fun limpiar() {
        nombre = ""; apellido = ""; email = ""
        telefono = ""; edad = ""
        nombreError = false; apellidoError = false
        emailError = false; telefonoError = false; edadError = false
    }

    // ── FUNCIÓN VALIDAR ────────────────────────────
    fun validar(): Boolean {
        var valido = true
        if (nombre.isBlank())   { nombreError   = true; valido = false }
        if (apellido.isBlank()) { apellidoError = true; valido = false }
        if (email.isBlank() || !android.util.Patterns.EMAIL_ADDRESS
                .matcher(email).matches()) {
            emailError = true; valido = false
        }
        val n = edad.toIntOrNull()
        if (n == null || n !in 1..120) { edadError = true; valido = false }
        return valido
    }

    // ── FUNCIÓN GUARDAR ────────────────────────────
    fun guardar() {
        if (!validar()) return
        personas.add(Persona(nextId++,
            nombre.trim(), apellido.trim(),
            email.trim().lowercase(),
            telefono.trim(), edad.toInt()))
        limpiar()
    }

    // ── SCAFFOLD (estructura principal) ───────────
    Scaffold(
        snackbarHost = { SnackbarHost(snackbarHostState) },
        topBar = {
            TopAppBar(
                title = { Text("Registro de Personas") },
                colors = TopAppBarDefaults.topAppBarColors(
                    containerColor = MaterialTheme.colorScheme.primaryContainer
                )
            )
        }
    ) { padding ->
        LazyColumn(
            modifier = Modifier.fillMaxSize().padding(padding).padding(16.dp),
            verticalArrangement = Arrangement.spacedBy(8.dp)
        ) {
            item {
                /* ── Card del formulario ── */
                Card(shape = RoundedCornerShape(16.dp)) {
                    Column(Modifier.padding(16.dp)) {
                        OutlinedTextField(
                            value = nombre,
                            onValueChange = { nombre = it; nombreError = false },
                            label = { Text("Nombre *") },
                            isError = nombreError,
                            modifier = Modifier.fillMaxWidth(), singleLine = true
                        )
                        Spacer(Modifier.height(8.dp))
                        OutlinedTextField(
                            value = apellido,
                            onValueChange = { apellido = it; apellidoError = false },
                            label = { Text("Apellido *") },
                            isError = apellidoError,
                            modifier = Modifier.fillMaxWidth(), singleLine = true
                        )
                        Spacer(Modifier.height(8.dp))
                        OutlinedTextField(
                            value = email,
                            onValueChange = { email = it; emailError = false },
                            label = { Text("Correo *") },
                            isError = emailError,
                            modifier = Modifier.fillMaxWidth(), singleLine = true,
                            keyboardOptions = KeyboardOptions(keyboardType = KeyboardType.Email)
                        )
                        Spacer(Modifier.height(8.dp))
                        Row(Modifier.fillMaxWidth(), horizontalArrangement = Arrangement.spacedBy(8.dp)) {
                            OutlinedTextField(
                                value = edad,
                                onValueChange = { edad = it; edadError = false },
                                label = { Text("Edad") }, isError = edadError,
                                modifier = Modifier.weight(1f), singleLine = true,
                                keyboardOptions = KeyboardOptions(keyboardType = KeyboardType.Number)
                            )
                            OutlinedTextField(
                                value = telefono,
                                onValueChange = { telefono = it },
                                label = { Text("Teléfono") },
                                modifier = Modifier.weight(1f), singleLine = true
                            )
                        }
                        Spacer(Modifier.height(16.dp))
                        Button(onClick = { guardar() }, modifier = Modifier.fillMaxWidth(),
                            shape = RoundedCornerShape(50)) {
                            Text("Guardar Registro")
                        }
                        Spacer(Modifier.height(8.dp))
                        OutlinedButton(onClick = { limpiar() }, modifier = Modifier.fillMaxWidth(),
                            shape = RoundedCornerShape(50)) {
                            Text("Limpiar")
                        }
                    }
                }
            }

            items(personas, key = { it.id }) { persona ->
                PersonaCard(persona = persona, onEliminar = { personas.remove(persona) })
            }
        }
    }
}
